📝 1. L'inscription — Quels champs remplir ?

Lors de votre inscription, vous devez renseigner les informations suivantes. Certains champs sont obligatoires, d'autres sont optionnels mais nécessaires pour recevoir vos paiements.

Identité

Prénom & Nom Obligatoire
Votre identité civile, telle qu'elle apparaît sur vos documents officiels.
Ex : Jean Martin
Email Obligatoire
Adresse email valide — vous recevrez les factures Stripe sur cet email.
Ex : jean.martin@email.com
Téléphone Obligatoire
Numéro de téléphone français ou international (9 à 15 chiffres).
Ex : 0612345678 ou +33612345678
Mot de passe Obligatoire
Minimum 4 caractères. Choisissez un mot de passe sécurisé.

Informations personnelles

Date de naissance Optionnel
Format : JJ/MM/AAAA
Ex : 15/03/1990
Lieu & Pays de naissance Optionnel
Utilisés pour les vérifications KYC (anti-fraude).

Véhicule

Type de véhicule Obligatoire
Berline (3 passagers max) ou Van (7 passagers max).
Marque, Modèle, Couleur Optionnel
Ces informations sont transmises à l'ambassadeur pour identifier votre véhicule.
Ex : Peugeot 508 Noir
Immatriculation Optionnel
Format français uniquement : 2 lettres - 3 chiffres - 2 lettres.
Ex : AB-123-CD

Facturation (Stripe)

⚠️ Important : Ces champs sont nécessaires pour que SÉSAME puisse prélever les frais de service. Sans IBAN valide, votre compte sera suspendu lors du premier cycle de facturation.
SIRET Obligatoire
Numéro d'identification de votre entreprise (14 chiffres). En tant que chauffeur VTC, vous devez obligatoirement être enregistré comme auto-entrepreneur ou société. Le SIRET est sur votre Kbis ou votre attestation INSEE.
Ex : 73282932000074 (14 chiffres)
IBAN Obligatoire
Numéro de compte bancaire international. Stripe utilisera cet IBAN pour prélever les frais SÉSAME (20% de votre CA). L'IBAN se trouve sur votre RIB.
Ex : FR76 3000 6000 0112 3456 7890 189
✅
En fournissant votre IBAN, vous acceptez le mandat SEPA qui autorise SÉSAME à prélever automatiquement les frais de service sur votre compte bancaire.

💳 2. Comment fonctionne la facturation Stripe ?

SÉSAME utilise Stripe pour facturer les chauffeurs. Le client paie directement le chauffeur (espèces ou carte) — SÉSAME ne touche pas cet argent. SÉSAME facture uniquement ses frais de service (20% du CA) via Stripe.

Le flux d'argent

👤 Client paie 50€ au chauffeur
→
🚗 Chauffeur encaisse 50€
→
📄 Stripe facture 10€ à SÉSAME
→
🏦 Stripe prélève 10€ sur l'IBAN
→
💰 10€ → Compte SÉSAME

Le cycle de facturation hebdomadaire

Lundi 01h00
Stripe génère automatiquement la facture du chauffeur (20% du CA de la semaine) et tente le prélèvement SEPA sur son IBAN.
Lundi 13h00
Si la facture est impayée, Stripe envoie un email de relance au chauffeur avec un lien de paiement manuel.
Mardi 01h00
Si toujours impayée, le compte chauffeur est suspendu. Il ne peut plus recevoir de courses.
Après paiement
Dès que le paiement est confirmé par Stripe (webhook), le compte est réactivé instantanément.

❌ 3. Que se passe-t-il avec un mauvais IBAN ?

Situation Ce qui se passe Solution
IBAN invalide (format incorrect) Refusé dès l'inscription — impossible de créer le compte Vérifier le RIB et saisir l'IBAN correct
IBAN valide mais mauvais compte Prélèvement échoue le lundi — relance envoyée Corriger l'IBAN dans Profil → Modifier mon profil
Solde insuffisant Prélèvement échoue — Stripe réessaie automatiquement 3 fois S'assurer que le compte a les fonds suffisants
Compte suspendu après non-paiement Ne peut plus recevoir de courses Payer via le lien Stripe reçu par email
🔴
Les frais SÉSAME ne s'affichent jamais dans l'application chauffeur. Ils n'apparaissent que sur les factures Stripe envoyées par email. C'est une règle légale pour éviter toute ambiguïté avec un contrat de travail.

✏️ 4. Comment corriger son IBAN ?

Si vous avez saisi un mauvais IBAN, vous pouvez le corriger à tout moment depuis l'application :

  1. Ouvrir l'application SÉSAME Chauffeur
  2. Aller dans l'onglet Profil (en bas de l'écran)
  3. Descendre jusqu'à la section "Modifier mon profil"
  4. Corriger le champ IBAN
  5. Appuyer sur "Enregistrer"
  6. Contacter SÉSAME pour relancer le prélèvement
⚠️
Après correction de l'IBAN, le prélèvement ne sera pas automatiquement relancé. Vous devez contacter SÉSAME à support@sesame-pro.com ou payer directement via le lien Stripe reçu par email.

🏦 5. Le mandat SEPA — Ce que vous autorisez

En fournissant votre IBAN lors de l'inscription, vous signez un mandat SEPA (prélèvement automatique). Voici ce que ça signifie :

Ce que vous autorisez ✅

  • SÉSAME à prélever les frais de service (20% du CA) chaque semaine
  • Stripe à débiter votre IBAN au nom de SÉSAME
  • Réception des factures par email (jamais dans l'app)

Ce que SÉSAME ne peut PAS faire ❌

  • Prélever plus que les 20% du CA réalisé
  • Prélever sans facture générée par Stripe
  • Accéder à vos autres informations bancaires

🔄 6. Les relances automatiques Stripe

Si un prélèvement échoue (IBAN incorrect, fonds insuffisants, etc.), Stripe relance automatiquement :

Tentative Délai Action
Tentative 1 Lundi 01h00 Premier prélèvement SEPA
Tentative 2 +3 jours Nouvelle tentative automatique
Tentative 3 +5 jours Dernière tentative automatique
Échec final Après 3 échecs Facture impayée + suspension compte
